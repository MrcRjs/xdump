# xdump

Install
=========

```
$ mkdir -p ~/.share/bin
$ git clone https://github.com/MrcRjs/xdump ~/.share
$ chmod +x ~/.share/xdump/xdump.sh
$ ln ~/.share/xdump/xdump.sh ~/.share/bin
$ echo 'PATH=$PATH:~/.share/bin' > ~/.bash_profile
$ source ~/.bash_profile
```

Usage
=====

```
$ xdmup non-xemu-file.iso
```
